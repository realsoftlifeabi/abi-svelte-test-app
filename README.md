# Svelte User Management Dashboard

A simple User Management Dashboard built with Svelte + Vite as part of a technical test.

## Features

- User table (Name, Email, Role, Actions)
- Add, Edit & Delete users
- Search by name or email
- Fully reactive using Svelte's built-in reactivity
- No external frameworks

## Bonus Enhancements

- TypeScript
- Global state using a Svelte store
- localStorage persistence
- Light/dark mode toggle
- Smooth Svelte transitions
